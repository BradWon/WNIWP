
## GIS
- 정보제공 및 분석 : 각 지점의 모든 데이터 중 사용자가 선택한 데이터로 분석할 수 있는 기능 (물정보포털)
- 검색 및 다운로드 : 선택한 영역의 제원정보 일괄 다운로드 (지하수)
- 검색결과 : 선택한 영역의 모든 지점 정보 확인 기능 (물환경)
- 검색결과 : 검색결과에서 위치정보를 같이 보여주는 기능 (USGS)
- GIS 기능 구현 방안 : 영역선택 잘 보이는 곳으로, 항목 중복 선택
- GIS 기능 구현 방안 : 면선택, 데이터 확인, 결과, 데이터 다운로드

데이터 통합 : 선택한 데이터 비교 조회, 그래프 기간 빠른 설정(1일, 1주일 등), 대용량 데이터셋 연결, API 연결, 보고서 연결
다중 레이어 : 주제별, 측정항목별, 유역·행정별, 데이터 선택, 주제도 선택, 기본도 선택, 배경맵 선택, 초기화
검색 다변화 : 주소검색, 영역 검색
공간분석 : GIS 매쉬업, 지도에서 직접 선택(행정구역별, 수계별) , 방향 설정



## 콘텐츠
- 대국민 친화적인 다양한 콘텐츠 발굴 (물정보포털)
- 콘텐츠 큐레이션 (가뭄)

큐레이션 : 사용자의 눈높이에 맞는 콘텐츠 제공
콘텐츠 발굴 : 사용자 이해를 돕기 위해 물 관련 정책, 운영현황 및 교육 자료 등 다양한 방법을 통해 지속적으로 콘텐츠 발굴
다양한 콘텐츠 : 웹툰, 여행코스, 축제, 취미생활, 교육, 게임 등 사용자들에게 풍성한 물 정보 제공 필요



## 데이터 통합
- 데이터 : 물환경정보시스템 : 5종 (측정, 기타자료), WAMIS : 84종 (통계자료), 총 89종 추가
- 검색조건 : 데이터 개수 표시, 데이터 depth 표시
- 검색결과 : 데이터 정의 및 설명, 대용량 데이터셋 바로가기, OPEN API 바로가기



## 대용량 다운로드
- 데이터 검색 : 다운받을 데이터 미리 보기 (공공데이터포털)
- 비동기처리 (기상자료개방포털)

데이터 유통 : 데이터가 실제 필요한 사용자들(국민, 전문가, 연구자, 이해관계자)이 쉽게 다운로드
비동기처리 : 데이터 접근성을 비동기 기술로 개방, 데이터를 대용량으로 연도별 데이터 셋 구분, 사용자가 필요한 연도의 데이터 셋만 선택해 다운로드할 수 있도록 제공
대용량 다운로드 : 유역 내의 모든 지점에 대한 측정항목 정보를 제한 없이 통합적으로 한꺼번에 다운로드, 데이터 기간, 개별 측정망, 유역단위, 권역단위, 항목 등으로 조회
