
# RPF 메뉴별 구분
## <span style="color:#ea4335;">GIS</span>
- [ ] 현재 측정망별 자료 조회에서 유역 및 행정구역 단위의 모든 측정망 자료를 한 번에 조회할 수 있는 통합조회 기능
- [ ] 원하는 모든 지점에 대해 측정항목 정보를 한꺼번에 다운로드할 수 있는 기능 구현
- [ ] 데이터 유역단위 조회
  - 대상 : 한강, 낙동강, 금강, 섬진강, 영산강
  - 대상 항목 : 수질, 수문기상, 수생태, 지하수, 유역, 상하수도, 홍수, 가뭄
  - 지도기반 데이터 표출
- [ ] 데이터 권역단위 조회
  - 대상 : 대권역, 중권역, 소권역, 표준유역
  - 대상 항목 : 수질, 수문기상, 수생태, 지하수, 유역, 상하수도, 홍수, 가뭄
  - 지도기반 데이터 표출
- [ ] 데이터 기간 조회
- [ ] 데이터 항목별 조회
- [ ] 데이터 기관별 조회
  - 국립환경과학원 데이터 조회/표출 : 하천 수질, 호소 수질, 보 수질, 수 생태, 조류
  - 홍수통제소 데이터 조회/표출 : 수문기상, 유역, 하천, 댐, 이수, 홍수
  - 한국환경공단 데이터 조회/표출 : 하수도정보, TMS
  - 수자원공사 데이터 조회/표출 : 댐/보 수문자료, 기상자료, 용수, 상수도, 지하수, 가뭄
  - 환경부 데이터 조회/표출 : 하천, 유역
  - 기타 기관 데이터 조회/표출 : 기타
- [ ] 기본기능 지원(SFR-033)
  - 전체화면, 초기화, 저장하기, PDF로 다운로드, JPG로 다운로드, SHP로 다운로드, 인쇄, 튜토리얼(체험하기)
- [ ] 지리정보 설정 기능
  - 일반지도, 위성지도, 백지도 등 선택
  - 지도확대, 지도축소, 투명도 설정, 지도라벨 색상설정, 지도라벨 폰트설정
  - 지도 배경 설정, 사용자 관심지역 등록/수정/삭제 등
----
- [ ] 데이터 통합 : 선택한 데이터 비교 조회, 그래프 기간 빠른 설정(1일, 1주일 등), 대용량 데이터셋 연결, API 연결, 보고서 연결
- [ ] 다중 레이어 : 주제별, 측정항목별, 유역·행정별, 데이터 선택, 주제도 선택, 기본도 선택, 배경맵 선택, 초기화
- [ ] 검색 다변화 : 주소검색, 영역 검색
- [ ] 공간분석 : GIS 매쉬업
- [ ] 지도에서 직접 선택(행정구역별, 수계별) , 방향 설정
---
- [ ] 표출하고 싶은 항목들만(관리측정망, 오염측정망 등)을 동시에 표출하고 싶음
- [ ] 측정망 지점 검색 기능 추가 요청
- [ ] 팝업창 스크롤 시 상단에 지점명칭 고정 표출 필요
- [ ] 조회기간 시작시점 선택 후 기간(예: 1주일, 1개월 등)으로 종료시점 설정 가능했으면 좋겠음


## <span style="color:#ea4335;">주제별 검색</span>
- [ ] 데이터 : 물환경정보시스템 : 5종 (측정, 기타자료), WAMIS : 84종 (통계자료), 총 89종 추가
- [ ] 검색조건 : 데이터 개수 표시, 데이터 depth 표시
- [ ] 검색결과 : 데이터 정의 및 설명, 대용량 데이터셋 바로가기, OPEN API 바로가기
---
- [ ] 키보드로 조회기간 입력 가능했으면 좋겠음
- [ ] 선택 가능한 조회기간을 현재시점까지만 표출했으면 좋겠음
- [ ] 조회기간 시작시점 선택 후 기간(예: 1주일, 1개월 등)으로 종료시점 설정 가능했으면 좋겠음 


## <span style="color:#ea4335;">대용량 데이터셋</span>
- [ ] 유역 내의 모든 지점에 대해 측정항목 정보를 한꺼번에 다운로드할 수 있는 체계로 개선
- [ ] 유역 기반의 데이터 셋을 제공하여 원하는 정보를 측정망 제한, 항목 제한, 기간 제한 없이 통합적으로 제공할 수 있도록 개선
- [ ] 데이터 기간 조회
- [ ] 개별 측정망 조회
  - 유역별, 권역별, 관할기관별
- [ ] 데이터 유역단위 다운로드
  - 대상 : 한강, 낙동강, 금강, 섬진강, 영산강, 제주도
  - 대상 항목 : 수질측정망 데이터 다운로드
- [ ] 데이터 권역단위 다운로드
  - 대상 : 대권역, 중권역, 소권역, 표준유역
  - 대상 항목 : 수질측정망 데이터 다운로드
- [ ] 항목별 다운로드
  - 대상 항목 : 수질, 수문기상, 수생태, 지하수, 유역, 상수도, 하수도, 홍수, 가뭄, 녹조
---
- [ ] 데이터 검색 : 다운받을 데이터 미리 보기 (공공데이터포털)
- [ ] 비동기처리 : 데이터 접근성을 비동기 기술로 개방, 데이터를 대용량으로 연도별 데이터 셋 구분, 사용자가 필요한 연도의 데이터 셋만 선택해 다운로드할 수 있도록 제공
- [ ] 대용량 다운로드 : 유역 내의 모든 지점에 대한 측정항목 정보를 제한 없이 통합적으로 한꺼번에 다운로드
- [ ] 데이터 기간, 개별 측정망, 유역단위, 권역단위, 항목 등으로 조회
